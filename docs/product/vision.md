# Vision — ServoLink / 舵连设计台

## 项目名称
- 中文名：舵连设计台
- 英文名：ServoLink

## 核心寓意
"舵连"取舵面与连杆之意，直观表达项目的核心物理对象。
"设计台"表达这是一个供设计师反复实验、调整、优化的数字工作台。

## 项目愿景
打造一个能让航模爱好者快速完成舵面连杆机构选型、计算、评估与优化的桌面工具，
把原先需要笔算+Excel+反复试凑的痛苦流程压缩为几分钟的参数调整。

## 最终目标（North Star）
1. **算得准**：运动学正反解结果精确到小数点后 1 位（角度°）
2. **评得快**：200 点全范围扫描 < 0.1 秒，传动质量一目了然
3. **优得对**：网格搜索 + 局部细化在 5 秒内给出 Top 10 推荐方案
4. **看得清**：机构动画 + 曲线图帮助直观理解运动规律
5. **学得懂**：完整的机构学理论文档 + 工程笔记，既是工具也是教材

## 核心价值方向
1. 把几何直觉转化为数值结果
2. 把试凑过程自动化
3. 把工程约束（传动角、死点、线性度）量化可视
4. 把设计决策可追溯、可复现（ADR + JSON 存储）

## 非目标
1. 不做 3D 渲染（2D 机构简图足够）
2. 不做有限元分析（力学由舵机参数表覆盖）
3. 不做实时闭环控制仿真
4. 不追求移动端 / Web 端（桌面工具即可）
