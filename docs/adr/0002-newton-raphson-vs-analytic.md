# ADR 0002: Use Newton-Raphson iteration for forward kinematics

## Status
Accepted

## Context
舵面连杆机构的正解问题：已知伺服臂转角 θ₁，求舵角 θ₂。

数学上，四杆机构的闭环方程可以化为关于 θ₂ 的一元方程：
$$F(\theta_2) = |B - A|^2 - L^2 = 0$$

该方程的解析解存在（可化为 $\cos(\theta_2 + \psi) = \text{const}$ 的形式），但需要分支判断（开式/闭式装配），且公式冗长易错。

## Decision
使用 **Newton-Raphson 迭代法** 作为正解算法。

## Reasons
1. **解析解不具通用性**：扩展至多连杆机构时解析解不存在，迭代法可复用框架
2. **解析解分支复杂**：需要根据 $\sin\theta_2$ 的正负判断装配模式，比初始猜测更易出错
3. **Newton-Raphson 收敛极快**：在舵面连杆机构的典型工作区（远离奇异位形），2-5 次迭代即可收敛到 1e-10 精度
4. **教学价值**：迭代法是数值分析核心概念，理解其原理对系统工程师有长远价值

## Implementation Details
- 解析求导 `dF/dθ₂`（非数值差分），保证迭代稳定性
- 初值用余弦定理几何估算（O₂-A-B 三角形）
- 步长阻尼 0.5 rad（防止跨越分支）
- 最大 50 次迭代，容限 1e-10（残差）+ 1e-8（角度）
- 收敛失败返回 `std::nullopt`

## Alternatives Considered
- **解析解**：表达式长、需要大量分支判断
- **数值差分**：简单但精度差、计算量大
- **fsolve 黑箱库**：失去控制权和教学价值

## Consequences
1. 在极端参数下可能不收敛（已通过测试覆盖边缘情况）
2. 反解仍使用几何初值+正解验证策略
